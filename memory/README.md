# Figlets Project Memory

This folder is for agent continuity, not public product documentation.

- `PROJECT_MEMORY.md` is the active implementation history: recent shipped behavior, recovery context, stale-session notes, and handoff details that help a future developer agent resume work quickly.
- Stable architectural or product decisions live in [`../docs/decisions.md`](../docs/decisions.md).
- Public setup, usage, and release guidance should live in the root `README.md` or `docs/`, not in this folder.

Designer-facing agents should not read this folder before starting the Figlets designer flow. Developer agents should read it alongside `docs/decisions.md` before editing the repository.
